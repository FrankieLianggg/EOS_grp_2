# JDBC Class Library Project - EOS_grp_2

## Overview
This project implements a reusable JDBC class library using Java and Microsoft JDBC Driver for SQL Server.

The library abstracts database operations and applies design patterns such as the Abstract Factory pattern.

## Features
- Database connection management
- Query execution
- DAO abstraction layer
- Transaction support
- Swing-based UI demo

## Technologies
- Java
- JDBC (Microsoft SQL Server Driver)
- SQL Server / Azure Data Studio

## Setup Instructions
1. Install Java JDK 8+
2. Install SQL Server or use Azure Data Studio
3. Add JDBC driver to classpath or Maven
4. Update DbConfig.java credentials
5. Run MainUI.java

## Design Pattern Used
- Abstract Factory Pattern for DAO creation
